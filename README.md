# 구독의 민족 | Web App by React.js + Node.js 

mongoDB: 추가 연동 예정

## 개발 툴

Visual Studio Code (VS Code)

## 개발 환경 설정

### `npm i`

프로젝트를 클론 받고, vs code 터미널에 npm i 입력해서 개발 시작 환경 세팅

## 서버 실행

### `npm run start`

포트 3000번으로 서버 실행 (개발 모드)

+ http://localhost:3000
+ 프로젝트 내 파일을 저장할 때마다 서버가 새로 실행되어 수정사항이 자동 반영됨
+ vs code 터미널에서 에러 확인 가능
